    Quick Start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get your SSP Backend API running in minutes
  </p>

  <Steps>
    <Step title="Clone the repository">
      Clone the SSP Backend repository and install dependencies.
      
      ```bash
      git clone https://github.com/SSPEstadias/SSP-BACK.git
      cd SSP-BACK
      npm install
      ```
    </Step>
    
    <Step title="Configure environment variables">
      Create a `.env` file with your database credentials and JWT secret.
      
      ```bash
      DB_HOST=localhost
      DB_PORT=5432
      DB_USERNAME=your_username
      DB_PASSWORD=your_password
      DB_NAME=ssp_db
      JWT_SECRET=your_jwt_secret_key
      PORT=3000
      ```
      
      <Note>See the [Environment Variables](/guides/environment-variables) guide for all available configuration options.</Note>
    </Step>
    
    <Step title="Set up the database">
      Create your PostgreSQL database and run migrations.
      
      ```bash
      npm run seed:admin
      npm run start:dev
      ```
      
      <Tip>The seeding script will create an initial admin user for you to get started.</Tip>
    </Step>
    
    <Step title="Authenticate and make your first request">
      Login to get a JWT token and make your first API call.
      
      ```bash
      curl -X POST http://localhost:3000/auth/login \
        -H "Content-Type: application/json" \
        -d '{"nomUsuario":"admin","contrasena":"your_password"}'
      ```
      
      <Accordion title="Example response">
        ```json
        {
          "access_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
          "user": {
            "id": 1,
            "nombre": "Admin User",
            "rol": "Admin"
          }
        }
        ```
      </Accordion>
    </Step>
  </Steps>
</div>
