---
title: "Update Actividad"
description: "Update an existing activity"
---

Updates one or more fields of an existing activity.

## Endpoint

```bash
PATCH /actividades/:id
```

## Authentication

This endpoint requires a valid JWT token.

```bash
Authorization: Bearer YOUR_JWT_TOKEN
```

## Path Parameters

<ParamField path="id" type="number" required>
  The unique identifier of the activity to update
</ParamField>

## Request Body

All fields are optional. Only include fields you want to update.

<ParamField path="nombre" type="string">
  Name of the activity
</ParamField>

<ParamField path="descripcion" type="string">
  Description of the activity
</ParamField>

<ParamField path="objetivo" type="string">
  Objective or goal of the activity
</ParamField>

<ParamField path="categoria" type="string">
  Activity category. Must be one of:
  - `TRABAJO_COMUNITARIO`
  - `LIDERAZGO_COMUNITARIO`
  - `ATENCION_SUSTANCIAS`
  - `EDUCACION_PARA_LA_VIDA`
  - `PROMOCION_CULTURAL_DEPORTIVA`
</ParamField>

## Response

Returns the updated activity object.

<ResponseField name="id" type="number">
  Unique identifier for the activity
</ResponseField>

<ResponseField name="nombre" type="string">
  Name of the activity
</ResponseField>

<ResponseField name="descripcion" type="string">
  Description of the activity
</ResponseField>

<ResponseField name="objetivo" type="string">
  Objective of the activity
</ResponseField>

<ResponseField name="categoria" type="string">
  Activity category
</ResponseField>

<ResponseField name="activo" type="boolean">
  Whether the activity is active
</ResponseField>

## Example Request

```bash
curl -X PATCH http://localhost:3000/actividades/1 \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "descripcion": "Actividad mejorada de limpieza y embellecimiento de áreas verdes",
    "objetivo": "Contribuir significativamente al mejoramiento del entorno comunitario"
  }'
```

## Example Response

```json
{
  "id": 1,
  "nombre": "Limpieza de parques públicos",
  "descripcion": "Actividad mejorada de limpieza y embellecimiento de áreas verdes",
  "objetivo": "Contribuir significativamente al mejoramiento del entorno comunitario",
  "categoria": "TRABAJO_COMUNITARIO",
  "activo": true
}
```

## Error Responses

### Not Found

```json
{
  "statusCode": 404,
  "message": "Actividad not found"
}
```

<Note>
  You only need to include the fields you want to update. Fields not included in the request will remain unchanged.
</Note>
