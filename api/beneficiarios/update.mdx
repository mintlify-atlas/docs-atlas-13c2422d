---
title: "Update Beneficiario"
description: "Update an existing beneficiario"
---

Updates one or more fields of an existing beneficiario.

## Endpoint

```bash
PATCH /beneficiarios/:id
```

## Authentication

This endpoint requires a valid JWT token.

```bash
Authorization: Bearer YOUR_JWT_TOKEN
```

## Path Parameters

<ParamField path="id" type="number" required>
  The unique identifier of the beneficiario to update
</ParamField>

## Request Body

All fields are optional. Only include fields you want to update.

<ParamField path="nombre" type="string">
  Full name of the beneficiario (max 150 characters)
</ParamField>

<ParamField path="tiempoAsignado" type="number">
  Assigned time amount (must be greater than 0)
</ParamField>

<ParamField path="unidadTiempo" type="string">
  Time unit. Must be one of:
  - `HORAS` - Hours
  - `MESES` - Months
</ParamField>

## Response

Returns the updated beneficiario object.

<ResponseField name="id" type="number">
  Unique identifier for the beneficiario
</ResponseField>

<ResponseField name="nombre" type="string">
  Full name of the beneficiario
</ResponseField>

<ResponseField name="fechaIngreso" type="string">
  Date of entry
</ResponseField>

<ResponseField name="tiempoAsignado" type="number">
  Assigned time amount
</ResponseField>

<ResponseField name="unidadTiempo" type="string">
  Time unit (HORAS or MESES)
</ResponseField>

<ResponseField name="creadoEn" type="string">
  Timestamp when record was created
</ResponseField>

## Example Request

```bash
curl -X PATCH http://localhost:3000/beneficiarios/1 \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "tiempoAsignado": 8,
    "unidadTiempo": "MESES"
  }'
```

## Example Response

```json
{
  "id": 1,
  "nombre": "Juan Pérez García",
  "fechaIngreso": "2026-03-01",
  "tiempoAsignado": 8,
  "unidadTiempo": "MESES",
  "creadoEn": "2026-03-01T10:00:00.000Z"
}
```

## Error Responses

### Not Found

```json
{
  "statusCode": 404,
  "message": "Beneficiario not found"
}
```

### Validation Error

```json
{
  "statusCode": 400,
  "message": [
    "La unidad de tiempo debe ser HORAS o MESES"
  ],
  "error": "Bad Request"
}
```

<Note>
  You only need to include the fields you want to update. Fields not included in the request will remain unchanged.
</Note>
