---
title: "Get Salud by Beneficiario"
description: "Retrieve health profile by beneficiario ID"
---

Retrieves the health profile associated with a specific beneficiario.

## Endpoint

```bash
GET /salud/beneficiario/:beneficiarioId
```

## Authentication

This endpoint requires a valid JWT token.

```bash
Authorization: Bearer YOUR_JWT_TOKEN
```

## Path Parameters

<ParamField path="beneficiarioId" type="number" required>
  The unique identifier of the beneficiario whose health profile you want to retrieve
</ParamField>

## Response

Returns the health profile object for the specified beneficiario.

<ResponseField name="id" type="number">
  Unique identifier for the health profile
</ResponseField>

<ResponseField name="beneficiarioId" type="number">
  ID of the associated beneficiario
</ResponseField>

<ResponseField name="esAptoFisico" type="boolean">
  Physical fitness status
</ResponseField>

<ResponseField name="padecEnfermedad" type="boolean">
  Medical condition status
</ResponseField>

<ResponseField name="nombreEnfermedad" type="string">
  Name of medical condition (if applicable)
</ResponseField>

<ResponseField name="consumeSustancias" type="boolean">
  Substance consumption status
</ResponseField>

<ResponseField name="tipoSustancias" type="string">
  Type of substances (if applicable)
</ResponseField>

<ResponseField name="afiliadoServicioSalud" type="string">
  Health service affiliation
</ResponseField>

<ResponseField name="necesitaLentes" type="boolean">
  Glasses requirement
</ResponseField>

<ResponseField name="observacionesMedicas" type="string">
  Medical observations
</ResponseField>

<ResponseField name="fechaActualizacion" type="string">
  Last update timestamp
</ResponseField>

## Example Request

```bash
curl -X GET http://localhost:3000/salud/beneficiario/1 \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

## Example Response

```json
{
  "id": 1,
  "beneficiarioId": 1,
  "esAptoFisico": true,
  "padecEnfermedad": false,
  "nombreEnfermedad": null,
  "consumeSustancias": false,
  "tipoSustancias": null,
  "afiliadoServicioSalud": "IMSS",
  "necesitaLentes": false,
  "observacionesMedicas": null,
  "fechaActualizacion": "2026-03-05T10:30:00.000Z"
}
```

## Error Responses

### Not Found

```json
{
  "statusCode": 404,
  "message": "Health profile not found for this beneficiario"
}
```

<Note>
  This endpoint is useful when you know the beneficiario ID and need to quickly retrieve their health information without searching through all health profiles.
</Note>

<Info>
  To update the health profile for a specific beneficiario, use the [Update Salud by Beneficiario](/api/salud/update-by-beneficiario) endpoint.
</Info>
