---
title: "List Salud Profiles"
description: "Retrieve all health profiles"
---

Retrieves a list of all health profiles in the system.

## Endpoint

```bash
GET /salud
```

## Authentication

This endpoint requires a valid JWT token.

```bash
Authorization: Bearer YOUR_JWT_TOKEN
```

## Response

Returns an array of health profile objects.

<ResponseField name="id" type="number">
  Unique identifier for the health profile
</ResponseField>

<ResponseField name="beneficiarioId" type="number">
  ID of the associated beneficiario
</ResponseField>

<ResponseField name="esAptoFisico" type="boolean">
  Physical fitness status
</ResponseField>

<ResponseField name="padecEnfermedad" type="boolean">
  Medical condition status
</ResponseField>

<ResponseField name="nombreEnfermedad" type="string">
  Name of medical condition
</ResponseField>

<ResponseField name="consumeSustancias" type="boolean">
  Substance consumption status
</ResponseField>

<ResponseField name="tipoSustancias" type="string">
  Type of substances
</ResponseField>

<ResponseField name="afiliadoServicioSalud" type="string">
  Health service affiliation
</ResponseField>

<ResponseField name="necesitaLentes" type="boolean">
  Glasses requirement
</ResponseField>

<ResponseField name="observacionesMedicas" type="string">
  Medical observations
</ResponseField>

<ResponseField name="fechaActualizacion" type="string">
  Last update timestamp
</ResponseField>

## Example Request

```bash
curl -X GET http://localhost:3000/salud \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```
